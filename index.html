pinkyGrid!

Why?

I was thinking of just making my own, simple and responsive CSS grid system, where I won't be forced to use numbers for the columns, but without numbers and they will be equal to me automatically. I mainly wanted to test myself in CSS if I was able to write something like this. That's how pinkyGrid was born!.

What?

Sometimes I needed to use a grid system for quick frontend projects, but I didn't want to use unnecessarily large third-party frameworks. I wanted something quick and small.

How to use?

It's simple. Download .css file of this framework, attach it in html header and use it like you can see below.
